# Imposter

Everybody gets a word but not the imposter, they will have to use their hint to try to figure it out.


## ❗Disclaimer❗

This is ONLY designed and optimized for mobile devices.


## ⭐ Roles 

- Innocent
- Imposter
- Jester
- Detective
- Lover
- Prosecutor


## Role specifics

### Innocent
- Find the imposter and vote them out.

### Imposter
- Figure out the word and deceive the other players.

### Detective
- Find the imposter and vote them out.
- Analysis will let you see all special roles for that round.

### Jester
- Cause chaos and get voted out.

### Prosecutor
- Get your target voted out.

### Lover
- Protect your lover, if they lose you do too.
